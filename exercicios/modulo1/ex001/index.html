Documento HTML: HyperText Markup Language

ATENÇÃO

Para iniciar um arquivo HTML, tem que ter as seguintes tag (No programa VS Code só inicia com o "!")

Exemplo:

<!DOCTYPE html>
<html lang="en"><---- Tem que mudar para "pt-br----">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document</title><--- Aqui muda o titulo do aquivo que aparecerá na aba--->
</head>
<body>


Entre os espaços Body conterá o conteúdo do site




</body>
</html>

E assim começa a ser estruturado o HTML de um site
