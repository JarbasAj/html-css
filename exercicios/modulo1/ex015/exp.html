CSS - locais ou Internos

Enquanto o estilo Inline é colocado as CSSs ou estilos da pagina em cada tag, nesta forma criamos um conceito que se aplica a toda a pagina ou comandos do HTML.

Logo abaixo da tag <title> dentro do comando <head> aplicamos os estilos CSS com a tag <style>

<style>
body{
background-color: lightsteelblue;
font-family: Arial, Helvetica,
sans-serif;
font-size: 15px;
}
h1{
color: darkred;
background-color: lightskyblue;
}
h2{
color: darkred;
}
p{
text-align: justify;
}
</style>

Assim, cada elemento pode ser feito dando o efeito em toda a pagina.
como:

  • body{
  • h1{
  • h2{
  • p{

    • Cada comando dentro da tag <style> será responsável pelo estilo.
