Colocando o favicon na aba

Como colocar um favicon?
Favicon é o icone que aparece na aba do navegador. Para isso precisa colocar
a tag link:favicon
<link rel="shortcut icon" href="favicon.ico" type="image/x-icon">

Na referencia "href=" Tem que fazer da mesma maneira
como é feito para colocar uma imagem
ctrl + space
