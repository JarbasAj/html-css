Usando Links externos

Exemplo de abertura de links em outra aba

Acessando uma pagina da internet, no caso será o meu Twitter

Para que um trecho do seu texto esteja linkado utilza-se as tags
<a href="url externa" target="_blank" rel="external">Trecho do texto a ser linkado</a>

As tags

<a>
Para iniciar a tag de marcação de hyperlink
target="_blank"
Para indicar que o link deverá ser aberto em outra aba
rel="external"
Para indicar aos mecanismo de busca que se trata de um link externo

Exemplo adicional

hreflang="idioma"
Permite indicar qual é o idioma principal do site para onde o link está desviando o fluxo de navegação.
Isso vai permitir avisar ao navegador e a softwares de tradução como lidar caso o visitante opte por traduzir
automaticamente os conteúdos.

Exemplo de abertura de link na mesma aba

Para acessa o link do youtube do meu Canal

Para links dentro do seu site utiliza-se essa tag:
<a>
<a href="url de dentro do seu site">trecho do texto a ser linkado</a>

Essa forma fecha a aba atual e abre uma nova aba por não ter as tags target e rel

Links para download

  • Baixar modulo em PDF
  • Modulo compactado .ZIP

Para colocar um link de download em seu site utiliza-se os parâmetros:
<a href="local onde está o arquivo" download="nome do arquivo" type="application/zip">Texto do link do download</a>

  • download="nome do arquivo"

    • Indica para o navegador fazer o download do arquivo

  • type="application/zip"

    • Indica ao navegador o tipo de download que será feito.
